Real-time Monitoring System

Unlike traditional offline lab testing, Eco-Spectrum achieves true "end-to-end" real-time response. Data collected by the autonomous boat is synchronized to the shore terminal in milliseconds via the 4G network.

  • Onboard Computing: Embedded platform processes spectral data directly.
  • Real-time Interaction: Shore terminals receive alerts instantly.
  • Visualization: Dynamic generation of water quality heatmaps.
System Test: Boat cruising & Shore data sync
